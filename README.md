# Homework Class Project

## Описание

Этот проект представляет собой реализацию классов для управления продуктами и категориями. Основные компоненты проекта включают классы `Product` и `Category`, которые позволяют создавать, управлять и отслеживать различные продукты и категории в системе.

## Классы

### Product

Класс `Product` представляет продукт с следующими атрибутами:

- `name` (str): Название продукта.
- `description` (str): Описание продукта.
- `price` (float): Цена продукта.
- `quantity` (int): Количество доступных единиц товара.

### Category

Класс `Category` используется для управления категориями продуктов и имеет следующие атрибуты:

- `name` (str): Название категории.
- `description` (str): Описание категории.
- `products` (list): Список продуктов в категории.

Кроме того, класс `Category` содержит два атрибута класса для отслеживания количества категорий и продуктов:

- `category_count` (int): Общее количество категорий.
- `product_count` (int): Общее количество продуктов.
